# 默认头像说明

## 文件位置
`miniprogram/images/default-avatar.png`

## 获取默认头像的方法

### 方法1：使用在线头像（推荐）

1. 访问以下网站下载免费头像：
   - https://www.flaticon.com/free-icons/user
   - https://www.iconfont.cn （搜索"用户头像"）
   - https://undraw.co/illustrations

2. 下载一张合适的头像图片
3. 命名为 `default-avatar.png`
4. 放到 `miniprogram/images/` 目录

### 方法2：使用 Emoji 表情

如果暂时没有图片，可以使用 emoji：

修改 `miniprogram/pages/profile/profile.wxml` 第 10 行：

```xml
<!-- 使用 emoji 作为默认头像 -->
<view class="avatar-emoji" wx:if="{{!userInfo.avatarUrl}}">👤</view>
<image 
  wx:else
  class="user-avatar" 
  src="{{userInfo.avatarUrl}}" 
  mode="aspectFill"
></image>
```

然后在 CSS 中添加样式：

```css
.avatar-emoji {
  width: 120rpx;
  height: 120rpx;
  border-radius: 50%;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 60rpx;
  color: #fff;
}
```

### 方法3：使用微信默认头像链接

```xml
<image 
  class="user-avatar" 
  src="{{userInfo.avatarUrl || 'https://thirdwx.qlogo.cn/mmopen/vi_32/POgEwh4mIHO4nibH0KlMECNjjGxQUq24ZEaGT4poC6icRiccVGKSyXwibcPq4BWmiaIGuG1icwxaQX6grC9VemZoJ8rg/132'}}" 
  mode="aspectFill"
></image>
```

## 推荐尺寸

- **最小**: 100x100px
- **推荐**: 200x200px
- **最佳**: 512x512px

## 当前状态

目前代码引用了 `/images/default-avatar.png`，如果该文件不存在，可能显示空白或错误图标。

建议：
1. 快速解决：使用方法3（微信默认头像链接）
2. 长期方案：准备一张品牌化的默认头像图片

## 临时解决方案

我已经在代码中使用了一个备用的网络图片链接。如果本地文件不存在，会尝试加载网络图片。
