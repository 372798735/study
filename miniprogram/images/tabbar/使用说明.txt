============================================
  快速获取图标（3种方式）
============================================

方式1：使用浏览器生成（最快）⭐⭐⭐
1. 双击打开 icon-generator.html 文件
2. 点击"一键下载全部图标"按钮
3. 将下载的 8 个 PNG 文件放入此文件夹
4. 完成！

方式2：从 iconfont 下载（最美观）⭐⭐
1. 访问 https://www.iconfont.cn/
2. 搜索并下载 home、question、video、user 图标
3. 下载两个版本（灰色 #7A7E83 和蓝色 #409EFF）
4. 重命名后放入此文件夹

方式3：查看详细指南
- 打开"图标获取指南.md"查看更多方式

============================================
  需要的图标文件（共8个）
============================================

✓ home.png          - 首页（未选中，灰色）
✓ home-active.png   - 首页（选中，蓝色）
✓ question.png      - 题目（未选中，灰色）
✓ question-active.png - 题目（选中，蓝色）
✓ video.png         - 视频（未选中，灰色）
✓ video-active.png  - 视频（选中，蓝色）
✓ profile.png       - 我的（未选中，灰色）
✓ profile-active.png - 我的（选中，蓝色）

============================================
  图标规格要求
============================================

尺寸：81px × 81px
格式：PNG（支持透明背景）
大小：每个文件 ≤ 40KB
颜色：
  - 未选中：#7A7E83（灰色）
  - 选中：#409EFF（蓝色）

============================================
  如何使用
============================================

1. 获取图标文件（使用上面的任一方式）
2. 将 8 个 PNG 文件放入此文件夹
3. 重启微信开发者工具
4. 查看底部导航栏，图标应该已经显示

============================================
  常见问题
============================================

Q：没有图标文件会怎样？
A：微信开发者工具会显示占位符，功能正常，只是没有图标

Q：图标不显示？
A：检查文件名是否正确，是否放在正确的文件夹

Q：想要更换图标？
A：直接替换对应的 PNG 文件即可
