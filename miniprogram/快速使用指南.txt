========================================
  微信小程序"我的"页面快速使用指南
========================================

问题：登录不了，获取不了头像和昵称
解决：已优化登录流程，提供两种登录方式

========================================
  方式一：快速登录（推荐）⭐⭐⭐
========================================

适用场景：
✅ 开发测试
✅ 无需后端
✅ 立即可用

使用步骤：
1. 打开"我的"页面
2. 点击"未登录"文字区域
3. 选择"快速登录"（左边按钮）
4. 阅读提示，点击"知道了"
5. 点击头像按钮，选择图片
6. 点击昵称，输入昵称
7. 完成！

========================================
  方式二：完整登录
========================================

前置条件：
⚠️ 需要后端服务运行

使用步骤：
1. 启动后端服务
   cd server
   npm run start:dev

2. 打开"我的"页面
3. 点击"未登录"文字区域
4. 选择"完整登录"（右边按钮）
5. 点击"继续"
6. 等待登录完成

========================================
  头像选择方法
========================================

步骤：
1. 确保已登录
2. 点击头像按钮（显示"点击选择头像"）
3. 选择图片：
   - 从相册选择
   - 或拍照
4. 确认
5. 看到"头像已更新"提示

注意：
❗ 要点击头像按钮，不是点击头像图片
❗ 必须先登录才能保存头像

========================================
  昵称编辑方法
========================================

步骤：
1. 确保已登录
2. 点击昵称输入框
3. 输入新昵称
4. 点击其他地方（失去焦点）
5. 看到"昵称已更新"提示

注意：
❗ 昵称不能为空
❗ 必须先登录才能保存昵称

========================================
  常见问题
========================================

Q: 头像选择后不显示？
A: 1. 先登录（快速登录即可）
   2. 再选择头像

Q: 昵称输入后不保存？
A: 1. 先登录
   2. 输入昵称后，点击其他地方

Q: 完整登录失败？
A: 1. 确认后端服务运行
   2. 或者使用快速登录

Q: 退出登录后还是已登录状态？
A: 工具 → 清除缓存 → 清除数据缓存

========================================
  调试技巧
========================================

1. 打开控制台（Console）
2. 查看日志输出：
   - "=== 我的页面加载 ===" 表示页面正常
   - "选择头像: ..." 表示头像选择成功
   - "输入昵称: ..." 表示昵称输入成功

3. 查看存储（Storage）
   - token: 登录令牌
   - userInfo: 用户信息
