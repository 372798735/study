info 设计说明与组件用途

微信小程序端设计要点

  • 首页：采用卡片式布局，突出学习进度和快捷功能入口
  • 题目练习：清晰的进度条和选项设计，支持单手操作
  • 题目讲解视频：每道题目可包含讲解视频，帮助学生理解
  • 视频学习：沉浸式播放体验，相关推荐增强学习连续性
  • 个人中心：简洁的统计展示和功能菜单
  • 底部导航：Material Design风格的Tab导航

PC管理后台设计要点

  • 登录页面：居中设计，品牌色彩突出
  • 控制台：数据可视化，一目了然的统计信息
  • 题目管理：表格化展示，支持筛选搜索和批量操作
  • 题目视频上传：每道题目可上传讲解视频，增强教学效果
  • 视频管理：卡片式布局，拖拽上传交互
  • 学生管理：学习进度可视化，状态标签清晰
  • 数据统计：图表化展示，关键指标突出
  • 系统设置：分类清晰，开关组件友好
  • 响应式设计：适配不同屏幕尺寸

Material Design 组件应用

Cards 卡片

用于内容分组和信息展示，提供清晰的视觉层次

FAB 浮动按钮

主要操作的快捷入口，提升用户操作效率

Navigation 导航

底部导航和侧边导航，适应不同设备特点
